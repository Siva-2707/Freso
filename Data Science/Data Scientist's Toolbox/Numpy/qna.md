## Introduction to NumPy

### What is the output of the following code?

```Python
import numpy as np

x = np.array([[3.2, 7.8, 9.2],
             [4.5, 9.1, 1.2]], dtype='int64')
print(x.itemsize)
```

Ans: **8**
